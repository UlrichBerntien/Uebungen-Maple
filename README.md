# Programmierübungen in Maple

Übungen mit dem Computermathematik-Programm [Maple](https://www.maplesoft.com/) Version 17.

In den Verzeichnissen liegen die Dateien (.mw) für Maple und die Ergebisse der
Rechnung in PDF-Dateien. Die PDF Dateien enthalten auch die Maple Befehle.
Maple kann auch PDF-Dateien erzeugen, die nur die Ergebnisse enthalten, also frei
vom Maple-Programmcode sind.
